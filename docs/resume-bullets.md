# BenchSight Resume Bullets

## Strong Project Bullets

- Built BenchSight, a GPU benchmarking and profiling workflow for evaluating custom CUDA kernels with structured timing, JSON result export, and a Streamlit comparison dashboard.
- Designed a repeatable benchmark methodology with warmup runs, CUDA event timing, p50 and p95 latency summaries, throughput tracking, and baseline-versus-optimized performance analysis.
- Implemented representative workloads including vector add, reduction, GEMM, Conv2D, and softmax, with NVTX instrumentation for Nsight Systems and Nsight Compute analysis.
- Created presentation-ready performance artifacts by connecting benchmark results, dashboard visualizations, and profiler evidence into one coherent workflow.

## Short Project Summary

BenchSight is a GPU benchmarking and profiling workflow that standardizes how custom CUDA workloads are measured, compared, visualized, and presented.
